## üìñ About the Project

**Sulawesi Explore** is a map-based web application (GIS) that aims to make it easier for tourists to find tourist destinations, culinary delights, and historical sites on Sulawesi Island. 

This project was created to address the lack of centralized information about local attractions. With interactive features, users can view locations on a map, leave reviews, and contribute by adding new ones.

This application uses modern technology **Next.js 15 (App Router)** with **Server Actions** for fast performance and optimal SEO.

## ‚ú® Main Features

### üåç For Visitors (Public)
1. **Interactive Map:** A dynamic map displaying pins of tourist locations.
2. **Smart Search & Filter:** Real-time search by name and category filters (Nature, Culinary, History).
3. **Detail Page:** A complete detail page with a photo gallery, description, amenities, and Google Maps routes.
4. **Rating & Review:** Visitors can leave stars and reviews without logging in (Guest Mode).
5. **Responsive Design:** Optimal display on mobile, tablet, and desktop.

### ‚ûï Contributors (Public)
1. **Add Listing:** Anyone can recommend new tourist attractions.
2. **Multi-Image Upload:** Upload multiple photos at once with a grid preview.
3. **Pin Point Location:** Determine coordinates (Latitude/Longitude) directly by clicking on the map.

### üõ°Ô∏è Admin Dashboard (Protected)
1. **Middleware Security:** The admin page is password-protected at the environment variable level.
2. **Approval System:** New listings are in `PENDING` status and must be approved by the Admin before going live.
3. **CRUD Management:** Admins can Edit (Update) and Delete (Delete) tour data.
4. **Overview:** View total statistics for listed tours.
